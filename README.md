# Stackly Baby Care Store

Stackly is a premium baby care web application offering the gentlest, safest, and most high-quality products for your little ones. 

## Live Demo
You can view the live site deployed on GitHub Pages here:
👉 **[Live GitHub Pages Website](https://lingalamanasa.github.io/Baby-Care-Store/)**

---

## Features
- **Responsive Layout**: Designed to look stunning on desktops, tablets, and mobile devices.
- **Product Showcase**: Curated categories and featured baby products (blankets, skincare lotion, onesies, teethers).
- **Meet Our Team**: Approaches and headshots of the company's founders and customer care team.
- **Interactive Location Map**: An embedded Google Maps view of the storefront on the contact page.
- **Fully Integrated Pages**:
  - [index.html](https://lingalamanasa.github.io/Baby-Care-Store/index.html) (Home)
  - [about.html](https://lingalamanasa.github.io/Baby-Care-Store/about.html) (About Us)
  - [services.html](https://lingalamanasa.github.io/Baby-Care-Store/services.html) (Services & FAQs)
  - [blog.html](https://lingalamanasa.github.io/Baby-Care-Store/blog.html) (Parenting Tips & Blog Articles)
  - [contact.html](https://lingalamanasa.github.io/Baby-Care-Store/contact.html) (Support Channels & Map)
  - [login.html](https://lingalamanasa.github.io/Baby-Care-Store/login.html) / [register.html](https://lingalamanasa.github.io/Baby-Care-Store/register.html) (User Access portal)
  - [customer-dashboard.html](https://lingalamanasa.github.io/Baby-Care-Store/customer-dashboard.html) / [admin-dashboard.html](https://lingalamanasa.github.io/Baby-Care-Store/admin-dashboard.html) (Simulated Dashboards)

## File Structure
All core style, scripting, and media resources are organized cleanly inside the `assets/` directory:
- [/assets/css/style_v3.css](https://github.com/lingalamanasa/Baby-Care-Store/blob/main/assets/css/style_v3.css) — Global stylesheets containing our warm pastel design system.
- `/assets/images/` — Compressed `.webp` assets, products, and team headshots.
- [/assets/js/main.js](https://github.com/lingalamanasa/Baby-Care-Store/blob/main/assets/js/main.js) — Client-side interaction controllers.
